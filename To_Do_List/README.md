# To-Do List

Command-line To-Do List application built with Python and SQLite3.

## Features
- Add tasks with title, description and automatic date
- View all tasks with their current state
- Update task state (e.g. not done → in progress → done)
- Delete tasks
- Data persists between sessions using local SQLite database

## How to use
1. Run the program
2. Choose an option from the menu:
   - `1` - Add task
   - `2` - See all tasks
   - `3` - Change task state
   - `4` - Delete task
   - `5` - Exit

## Requirements
- Python 3.x

## Run
```bash
python To_Do_List.py
```
