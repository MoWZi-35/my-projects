# Bubble Sort

Simple Bubble Sort algorithm implementation in Python.

## Features
- Sort 6 numbers from smallest to largest or largest to smallest
- Supports decimal numbers
- Option to sort again after each run

## How to use
1. Run the program
2. Enter 6 numbers
3. Choose sorting order:
   - `1` - smallest to largest
   - `2` - largest to smallest
4. See sorted result
5. Choose to sort again or exit

## Requirements
- Python 3.x

## Run
```bash
python buble_sort.py
```
