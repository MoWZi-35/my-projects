# 🎮 Rock, Paper, Scissors

Classic Rock, Paper, Scissors game played against the computer.

## Features
- 🤖 Computer opponent
- ✂️ Three choices: Rock (r), Paper (p), Scissors (s)
- 📊 Win/lose/draw tracking
- 🔄 Play multiple rounds
- ✅ Input validation

## Usage
```bash
python rock_paper_scissors.py
```
